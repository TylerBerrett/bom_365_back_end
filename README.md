# bom_365_back_end

GET https://bom-365.herokuapp.com/scripture/{MONTH}/{DAY} (MONTH must be the int of a month, DAY is a day of the month)

  "scriptures" : [ "Ether 1: 1-43", "1 and it", "2 came to pass", "Ether 2: 1-12", "1 after I", "2 was born" ]
}
